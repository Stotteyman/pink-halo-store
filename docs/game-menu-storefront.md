# Pink Halo Game Menu Storefront

## Objective

Create a shopping experience that feels like a cozy video game shop menu while still working as a normal ecommerce store.

## Experience Direction

Working name: Cozy Boutique RPG Menu.

The interface should use soft panels, rounded cards, friendly prompts, clear item grids, product images, prices, and quick shopping actions. The goal is to make browsing feel playful while keeping checkout easy.

## Main Interface

### Header

- Pink Halo logo
- Search
- Account or login
- Rewards balance
- Cart
- Help

### Categories

- New Arrivals
- Dresses
- Tops
- Lounge
- Accessories
- Sale
- Bundles
- VIP Drops

### Product Tiles

Each tile should show:

- Product image
- Product name
- Price
- Sale badge
- Stock badge
- Points earned

### Product Preview

The selected item panel should show:

- Large image
- Name
- Price
- Short description
- Size selector
- Color selector
- Quantity selector
- Points earned
- Add to Cart
- Buy Now
- Wishlist
- Shipping note
- Return note

## Account Features

Customers should be able to create an account, log in, save shipping addresses, view order history, save wishlist items, track rewards points, and use referral links.

## Rewards Features

Customers can earn points from purchases, referrals, approved reviews, account creation, email signup, birthday rewards, and VIP actions.

## Data Requirements

The game menu should use the same product, inventory, cart, order, customer, address, rewards, and referral data as the normal store.

## Accessibility

The interface should support keyboard use, screen readers, alt text, visible focus states, high contrast text, reduced motion, and a standard product grid fallback.

## Build Priority

1. Product and cart data.
2. Customer accounts and saved addresses.
3. Rewards and referrals.
4. Checkout.
5. Game menu presentation layer.
6. Animation and optional 3D boutique features.
7. Conversion testing against a standard product grid.
