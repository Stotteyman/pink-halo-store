# Pink Halo Documentation Operating System

This repo is the GitHub-side source for PinkHalo.co technical planning, admin tools, database architecture, and implementation notes.

Google Drive remains the business operating system for human-readable company documentation. GitHub is the technical execution system for code, schemas, issues, APIs, and developer-ready documentation.

## Sync Rule

Every major business document in Google Drive should have a matching technical or markdown reference in this repository when it affects the website, database, admin dashboard, automation, mockups, product operations, manufacturer workflows, shipping, fulfillment, or analytics.

## Core Areas

- `docs/admin-design-studio.md` — admin tools for creating designs and mockups.
- `docs/supply-chain-system.md` — manufacturer, order, inventory, fulfillment, and tracking workflows.
- `docs/database-architecture.md` — database entities and accuracy rules.
- `docs/google-drive-sync.md` — how Google Drive and GitHub stay aligned.
- `docs/product-pipeline.md` — idea-to-product-to-sale workflow.
- `docs/admin-dashboard.md` — admin dashboard requirements.

## Operating Principle

Pink Halo should not rely on fake dashboard numbers, manual guesses, or disconnected spreadsheets. Orders, revenue, inventory, shipping, returns, designs, manufacturers, and analytics should all be connected to real database records.
