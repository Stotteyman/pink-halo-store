# Admin Design Studio & Mockup System

Source Google Drive Doc: https://docs.google.com/document/d/19Bcbi0WG8N67VV9y1AhI0cfU591i4qZTvpVtoOt6JCA

## Purpose
Pink Halo should include an internal design studio where admins can create, edit, approve, and publish products and mockups.

## Design Workflow
Idea → Draft → Mockup → Internal Approval → Manufacturer Template → Product Listing → Launch.

## Admin Tools

- AI image generation
- Canva-style editor
- Layer editor
- Text and typography tools
- Color palettes
- Pattern library
- Template library
- Background remover
- Smart resizing
- Mockup generator
- Version history
- Approval queue

## Mockup Types
T-shirts, hoodies, sweatshirts, crop tops, mugs, tote bags, phone cases, packaging, social media ads, and lifestyle scenes.

## Asset Library
Fonts, icons, patterns, logos, graphics, photography, and seasonal templates.

## Database Tables
Designs, versions, mockups, collections, approval status, designer notes, launch dates, and performance metrics.

## Publishing Rules

- Nothing goes live until approved.
- Every design has version history.
- Every mockup is linked to products.
- Every product is linked to collections.

## Analytics
Views, add-to-cart rate, sales, conversion rate, revenue, returns, and customer reviews.
