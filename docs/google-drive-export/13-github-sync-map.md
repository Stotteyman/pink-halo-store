## Sync Rules
When Google Drive changes business process, update the matching GitHub technical document if it affects code, database, admin tools, analytics, automation, fulfillment, product publishing, or customer flow.

When GitHub changes code or data models, update Google Drive if it affects operations, SOPs, customer experience, product planning, or business workflow.

## Accuracy Rule
All dashboards, tracking, revenue, inventory, shipping, manufacturer metrics, design metrics, and order counts must come from the database. Do not rely on manual numbers, fake data, or disconnected spreadsheets in production.
