# START HERE - Pink Halo Operating System Index

Source Google Drive Doc: https://docs.google.com/document/d/1bszTsqRslpmUUiYio-5uujkJy__R8yWEx6d3XFqvW3o

## Purpose
This document is the front door for PinkHalo.co. The Google Drive should operate like the company’s internal business system, not just file storage. Every folder has a job, every document should help someone run the store, and every process should be simple enough for a future assistant, contractor, or employee to follow.

## Business Summary
Pink Halo is a women’s clothing and lifestyle boutique built around a dreamy soft-luxury identity: pink, halo, sparkle, confidence, feminine energy, and affordable boutique fashion. The store should sell women’s clothing, matching sets, dresses, loungewear, accessories, and seasonal drops through PinkHalo.co.

## Brand Direction
The brand should feel magical, feminine, soft, elegant, and modern. The official logo uses a glowing halo, pink script lettering, sparkle effects, and champagne-gold accents. This identity should guide the website, packaging, social media, email templates, product photography, and customer experience.

## Operating Rule
If a task repeats more than twice, create a system for it. If a customer asks the same thing more than twice, create a support script or FAQ. If a product sells well, create a repeatable sourcing and restock process. If a marketing post performs well, turn it into a content template.

## 90-Day Goal
Within 90 days, Pink Halo should have a complete brand identity, organized product strategy, working website plan, launch marketing calendar, supplier list, customer support process, and operating documentation strong enough to hand off to contractors.
